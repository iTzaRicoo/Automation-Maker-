# Dashboard Maker (Home Assistant Add-on)

Een visuele dashboard generator (Mushroom) die 2 dashboards maakt:
- **Simpel** (minder ruis, beginner-proof)
- **Uitgebreid** (meer detail)

## Installeren (als lokale add-on)
1. Kopieer de map `dashboard_maker` naar:
   `/addons/dashboard_maker/`
2. In Home Assistant: **Instellingen → Add-ons → Add-on Store → ⋮ → Repositories**
3. (Bij lokaal) herlaad, zoek **Dashboard Maker**, installeer en start.

## Gebruik
Open de add-on via **Open Web UI** (Ingress).
